# Dot Files (archived)

This repository is archived. Development continues at [yths/yths.dot-files](https://github.com/yths/yths.dot-files), where `nuunamnir` is now a first-class **theme preset name** rather than a separate user identity.

Content from this repo that was unique was migrated to the canonical home:

* Color-scheme rationale and palette table → [yths.dot-files/docs/notes.md](https://github.com/yths/yths.dot-files/blob/main/docs/notes.md)
* Color swatch PNGs → [yths.dot-files/assets/themes/nuunamnir/palette/](https://github.com/yths/yths.dot-files/tree/main/assets/themes/nuunamnir/palette)
* Post-base-install Arch walkthrough (display manager, plymouth, desktop environment, display calibration) → [yths.dot-files/docs/install.md](https://github.com/yths/yths.dot-files/blob/main/docs/install.md)

The qtile, web-greeter, plymouth, vscode, and rofi configurations were a strict subset of the yths counterparts and were not migrated separately.
